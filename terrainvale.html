Terrain Authority

Terrain Authority is the foundational terrain generation tool of the Vale Terrain Suite. Rather than attempting to simulate real-world geology, it creates believable environmental forms through artistic composition and controlled procedural generation.

Problem Solved

Many terrain generators overwhelm developers with complex simulations and extensive parameter systems. Terrain Authority focuses instead on the desired result: compelling terrain foundations created quickly while preserving artistic control.

Primary Features

  • Pressure-field based terrain generation
  • Parent and child regional terrain inheritance
  • Directional shelving for plateaus and stepped formations
  • Controlled ruggedness for natural visual breakup
  • Standardized Vale territory cadence
  • Rapid iteration with minimal setup

Requirements

  • Unity Editor
  • Unity Terrain System

Part of the VALE Workflow

Terrain Authority establishes the land upon which the remaining VALE tools operate.

  • Territory Authority
  • Path Authority
  • Ribbon Authority
  • Grade Authority
  • Paint Authority

Territory Authority

Territory Authority transforms large Unity terrains into standardized, interconnected territories designed for scalable world construction. Rather than treating an entire landscape as a single object, it establishes meaningful divisions that support organization, iteration, and future expansion.

Problem Solved

Large terrains become increasingly difficult to organize and maintain. Territory Authority converts existing Unity terrains into standardized territory grids while preserving visual continuity, making worlds easier to understand, expand, and manage.

Primary Features

  • Converts large terrains into standardized territory grids
  • Multiple territory subdivision configurations
  • Preserves terrain continuity through height reconstruction
  • Automatically establishes neighboring terrain relationships
  • Maintains visual cohesion across territory boundaries
  • Preserves the original terrain by disabling rather than destroying it

Requirements

  • Unity Editor
  • Unity Terrain System
  • Existing Unity Terrain

Part of the VALE Workflow

Territory Authority is the organizational foundation of the VALE workflow. It transforms completed terrain into structured territories that prepare the world for Path Authority, Ribbon Authority, Grade Authority, and Paint Authority.

Path Authority

Path Authority is a reusable path authoring system designed to establish meaningful movement throughout a world. Rather than painting roads directly onto terrain, it records intentional routes that can be preserved, extended, reused, and interpreted by other systems.

Problem Solved

Many worldbuilding workflows bind movement and appearance together, making routes difficult to revise or extend. Path Authority separates route definition from visual expression, preserving the truth of movement while allowing appearance to evolve independently.

Primary Features

  • Reusable path assets
  • Persistent path data storage
  • Support for branching and extending existing routes
  • Precise distance measurement
  • Author-defined route placement
  • Grade-aware path construction options
  • Designed for integration with Ribbon Authority
  • Supports meaningful travel calculations

Requirements

  • Unity Editor
  • Unity Terrain System
  • Recommended for use with Ribbon Authority

Part of the VALE Workflow

Path Authority establishes meaningful movement throughout the world. Terrain Authority creates the land, Territory Authority organizes it, and Path Authority defines how that world is traveled before Ribbon Authority gives those routes visible expression.

Ribbon Authority

Ribbon Authority transforms Path Authority data into visible environmental features. Rather than redefining routes, it expresses them. A single path may become a road, river, bridge, trench, aqueduct, lava flow, leyline, or any other linear feature that gives visual form to movement and connection.

Problem Solved

Many worldbuilding tools combine route creation and visual representation into a single step, making routes difficult to revise and experimentation costly. Ribbon Authority separates expression from intent, allowing the appearance to change while preserving the underlying path.

Primary Features

  • Generates visual features from existing Path Authority assets
  • Supports roads, rivers, bridges, trails, trenches, and other ribbon features
  • Preserves the original path data
  • Allows rapid visual experimentation
  • Grade-aware material support
  • Dynamic material assignment
  • Variable width controls
  • Supports animated materials and directional effects

Requirements

  • Unity Editor
  • Unity Terrain System
  • Existing Path Authority data

Part of the VALE Workflow

Ribbon Authority provides the visible expression of movement established by Path Authority. It transforms logical routes into believable environmental features while preserving the original path information for future revisions.

Grade Authority

Grade Authority is a terrain behavior authoring tool designed to establish meaningful environmental constraints through slope relationships rather than artificial restrictions. Instead of relying on invisible walls or guesswork, it allows designers to define the grades that govern possibility within their worlds.

Problem Solved

Modern games often rely on invisible barriers and inconsistent terrain sculpting to guide players. Grade Authority establishes believable environmental constraints through meaningful slope relationships, allowing the terrain itself to communicate what is possible.

Primary Features

  • Target grade authoring using meaningful slope values
  • Adjustable grade variance for natural results
  • Convergence-based terrain modification
  • Average grade measurement and feedback
  • Controlled approach toward desired outcomes
  • Support for traversal design and environmental storytelling
  • Reduces reliance on invisible barriers
  • Encourages natural environmental readability

Requirements

  • Unity Editor
  • Unity Terrain System
  • Existing Unity Terrain

Part of the VALE Workflow

Grade Authority teaches the environment what possibilities exist within the world. Following Terrain, Territory, Path, and Ribbon Authority, it establishes believable traversal before Paint Authority completes the world's visual identity.

Paint Authority

Paint Authority is a rule-based terrain texturing system designed to give environments visual voice through environmental relationships rather than repetitive manual painting. By combining height, slope, and painterly dry-brush techniques, it creates believable terrain transitions while preserving artistic intent.

Problem Solved

Manual terrain painting quickly becomes repetitive and inconsistent, especially across large worlds. Paint Authority applies textures through meaningful environmental rules, dramatically reducing repetitive work while preserving the artist's creative control.

Primary Features

  • Height-based texture rules
  • Slope-based texture rules
  • Painterly outer dry-brush placement
  • Painterly inner dry-brush placement
  • Natural transition generation
  • Rule-driven material distribution
  • World-wide terrain application
  • Reduced repetitive painting requirements
  • Preserves environmental readability

Requirements

  • Unity Editor
  • Unity Terrain System
  • Configured Terrain Layers
  • Existing Unity Terrain

Part of the VALE Workflow

Paint Authority is the final stage of the VALE Terrain workflow. After the land has been created, organized, connected, expressed, and refined through terrain grades, Paint Authority gives the completed world its visual voice through intelligent, rule-based terrain texturing.
