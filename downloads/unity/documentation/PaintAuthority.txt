PAINT AUTHORITY
INSTALLATION, MANUAL, LICENSE, AND SUPPORT

OVERVIEW
Paint Authority is part of the TerrainVale Suite for the Unity Editor.

Paint Authority is a rule-based terrain texturing system designed to give environments visual voice through environmental relationships rather than repetitive manual painting.

PROBLEM SOLVED
Manual terrain painting quickly becomes repetitive and inconsistent, especially across large worlds. Paint Authority applies textures through meaningful environmental rules.

REQUIREMENTS
- Unity Editor
- Unity Terrain System
- Configured Terrain Layers
- Existing Unity Terrain

INSTALLATION
1. Back up the Unity project before importing any editor extension.
2. Open the target project in a supported Unity Editor installation.
3. Import the supplied .unitypackage through Assets > Import Package > Custom Package.
4. Review the package contents, then complete the import.
5. Allow Unity to compile before opening the tool.
6. Use the tool's own creation commands to establish any required project folders or data structures.

PRIMARY FEATURES
- Height-based texture rules
- Slope-based texture rules
- Painterly outer dry-brush placement
- Painterly inner dry-brush placement
- Natural transition generation
- Rule-driven material distribution
- World-wide terrain application
- Reduced repetitive painting requirements
- Preserves environmental readability

WORKFLOW ROLE
Paint Authority is the final stage of the VALE terrain workflow. After the suite establishes land, territories, routes, visible features, and grades, Paint Authority gives the completed world its visual voice through rule-driven terrain texturing.

PACKAGE CONTENTS

USE AND PROJECT SAFETY
- Test the tool in a duplicate project or source-controlled branch before applying changes to production content.
- Save the project before terrain generation, conversion, painting, or destructive editor operations.
- Confirm terrain resolution, dimensions, layers, neighboring terrain relationships, and project settings before processing.
- Large terrains and high resolutions may require additional processing time and memory.
- Results depend on project configuration, Unity version, terrain data, and author-selected settings.
