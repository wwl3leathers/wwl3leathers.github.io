TALVAREN STUDIOS
RIBBON AUTHORITY
INSTALLATION, MANUAL, LICENSE, AND SUPPORT

OVERVIEW
Ribbon Authority is part of the TerrainVale Suite for the Unity Editor.

Ribbon Authority transforms Path Authority data into visible environmental features.

PROBLEM SOLVED
Many worldbuilding tools combine route creation and visual representation into a single step, making routes difficult to revise and experimentation costly.

REQUIREMENTS
- Unity Editor
- Unity Terrain System
- Existing Path Authority data

INSTALLATION
1. Back up the Unity project before importing any editor extension.
2. Open the target project in a supported Unity Editor installation.
3. Import the supplied .unitypackage through Assets > Import Package > Custom Package.
4. Review the package contents, then complete the import.
5. Allow Unity to compile before opening the tool.
6. Use the tool's own creation commands to establish any required project folders or data structures.

PRIMARY FEATURES
- Generates visual features from existing Path Authority assets
- Supports roads, rivers, bridges, trails, trenches, and other ribbon features
- Preserves the original path data
- Allows rapid visual experimentation
- Grade-aware material support
- Dynamic material assignment
- Variable width controls
- Supports animated materials and directional effects

WORKFLOW ROLE
Ribbon Authority provides the visible expression of movement established by Path Authority. It transforms reusable logical routes into roads, rivers, bridges, trails, trenches, and other environmental features without altering the original path data.

PACKAGE CONTENTS
- One self-contained Unity .unitypackage containing the tool code.

USE AND PROJECT SAFETY
- Test the tool in a duplicate project or source-controlled branch before applying changes to production content.
- Save the project before terrain generation, conversion, painting, or destructive editor operations.
- Confirm terrain resolution, dimensions, layers, neighboring terrain relationships, and project settings before processing.
- Large terrains and high resolutions may require additional processing time and memory.
- Results depend on project configuration, Unity version, terrain data, and author-selected settings.
