TALVAREN STUDIOS
PATH AUTHORITY
INSTALLATION, MANUAL, LICENSE, AND SUPPORT

OVERVIEW
Path Authority is part of the TerrainVale Suite for the Unity Editor.

Path Authority is a reusable path authoring system designed to establish meaningful movement throughout a world.

PROBLEM SOLVED
Many worldbuilding workflows bind movement and appearance together, making routes difficult to revise or extend. Path Authority separates route definition from visual expression.

REQUIREMENTS
- Unity Editor
- Unity Terrain System
- Recommended for use with Ribbon Authority

INSTALLATION
1. Back up the Unity project before importing any editor extension.
2. Open the target project in a supported Unity Editor installation.
3. Import the supplied .unitypackage through Assets > Import Package > Custom Package.
4. Review the package contents, then complete the import.
5. Allow Unity to compile before opening the tool.
6. Use the tool's own creation commands to establish any required project folders or data structures.

PRIMARY FEATURES
- Reusable path assets
- Persistent path data storage
- Support for branching and extending existing routes
- Precise distance measurement
- Author-defined route placement
- Grade-aware path construction options
- Designed for integration with Ribbon Authority
- Supports meaningful travel calculations

WORKFLOW ROLE
Path Authority establishes meaningful movement throughout the world. Terrain Authority creates the land, Territory Authority organizes it, and Path Authority defines how that world is traveled before Ribbon Authority gives those routes visible form.

USE AND PROJECT SAFETY
- Test the tool in a duplicate project or source-controlled branch before applying changes to production content.
- Save the project before terrain generation, conversion, painting, or destructive editor operations.
- Confirm terrain resolution, dimensions, layers, neighboring terrain relationships, and project settings before processing.
- Large terrains and high resolutions may require additional processing time and memory.
- Results depend on project configuration, Unity version, terrain data, and author-selected settings.
