Carriage Collection

$16.00

The Carriage Collection contains two fully assembled carriage prefabs—a traditional horse-drawn coach and a funeral hearse—along with the individual modular components used to construct them. Build complete vehicles in minutes or create your own custom variations by mixing and matching parts.

Every wheel, axle, body section, and decorative component has been designed for modular assembly. The included axles are prepared for wheel rotation scripts, making the collection suitable for both static scenes and moving vehicles.

Included Assets

  • Traditional horse-drawn carriage prefab
  • Funeral hearse prefab
  • Fully modular carriage components
  • Prepared wheel axles for animation scripts
  • Production-ready prefabs
  • Clean hierarchy organization

Requirements

  • Unity Editor
  • No additional packages required
